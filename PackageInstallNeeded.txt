KHÔNG CẦN CÀI DƯỚI ĐÂY NỮA
init -y(khơỉ tạo package.json)
npm install express(express.js)
npm install nodemon -D(hỗ trợ nodejs)
npm install mongoose(MongoDB)
npm install dotenv(truy cập các biến môi trường)
npm install(i) axios(call API)
npm install react-router
npm install cors(gộp địa chỉ local của backend và )
